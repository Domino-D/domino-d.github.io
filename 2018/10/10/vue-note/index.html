Vue学习笔记，随学随记

Vue 读音同 view

v-on: 事件绑定方法

缩写 @等价于 ‘v-on:’
举例：v-on:click=”todoRemove()” 等价于 @click=”todoRemove()”

v-for=”item of todos”

todos 是举例代表data中的todos数组数据，仅为举例。
vue给出的循环解决方案，数据双向绑定
vue特别推荐使用 of 语法，个人感觉更接近语义化。当然也兼容 in 语法。使用哪一个意义都一样，仅随个人喜好。

过滤结果的JS方法 .filter()

例如：xxx.filter(item => item > 0) 意为返回数组中大于0的所有值，并组成新的数组，作为返回对象
去除前导空格和末尾空格的JS方法（字符串中间的空格不会被删除） .trim()

v-model=”msg”

表单数据双向绑定，实现数据的实时更新并回流更新页面显示数据，表单验证

v-if=”set”

条件判别式，双向数据绑定，布尔值判别。若所传参数判定为true，所绑定标签在dom中生成正常事件流；若所传参数为false，所绑定标签的在渲染过程中，不生成事件流，最终在页面上也不可见

template 与 v-if 组合

vue 设定了内置对象 template ，实际结果中，是不可见的，不会产生真正的dom标签节点，可以用template来包裹需要操作的标签集合，通过将v-if绑定在template上，依据给定数据决定template及其内部包含元素的隐藏和展现功能，且不会在最终实际的dom中展现template标签
<template v-if="">content</template>

v-bind 与 class 属性配合，更改页面呈现样式可以实现多种类名的叠加显示效果，类同UI组件的显示效果

<p v-bind:class="todo.state"></p>

新建Vue

new一个新的vue对象，按照编写规范构造，el表示作用于哪一个文件，如 ‘index.html’，data标签下编写双向绑定数据，需要说明的是其中的数据并不仅限只读，还可以增删改查。

splice(param1,param2,item)

splice中文意思为绞接
param1 参数从0开始计算，第一个为0；param2参数为0，则不删除，不为0，则从param1指定的元素开始计算，向后删除param2个元素；item可以填写0~n个，表示补填内容。返回值为填充或者删除完成后的所有元素集合的数组

slice(param1,param2)

Slice中文意思为切下
Param1参数从0开始计算，第一个为0；param2参数为0，表示从param1开始计算到数组结束的所有元素被剥离出来，形成一个新的数组作为返回数组，param2不为0，表示从param1到param2所指的元素，全部剥离出来，形成一个新的数组作为返回数组。
