String 字符串

string - 字符串创建

  1. let str = 'abc' 字符串直接赋值

  2. let str = new String('abc') 新建字符串对象

string - 字符串操作

  1. charAt(index) 返回字符串中 index 指针对应位置下的字母值。index 从0开始计算。

  2. charCodeAt(index) 返回字符串中 index 指针对应位置下的字母的unicode数值。index 从0开始计算。

  3. concat(str2, str3, ... strn) 连接字符串。返回值为连接后的整个字符串。可以传多个参数。

    • 参数会被强制转换为字符串形式。对于数组类型的参数会被强制执行 join() ，转换后会默认保留 , 。

  4. indexOf(value, fromindex) lastIndexOf(value, fromindex)返回某个指定的字符串值在字符串中首次出现的位置。

    • fromindex 表示从给定字符串的那个指针开始找起；如果没有找到结果返回 -1 ；如果不指定，默认从头开始找，即从0开始。

    • 大小写敏感。

  5. match(value|reg) 返回数组对象，没有结果返回 null

    • 参数可以是字符串也可以是正则

      如果正则使用全局搜索 g ，则返回一个数组

    • 大小写敏感

  6. replace(value|reg, replacement) 返回替换结果字符串

    • 参数可以是字符串也可以是正则

    • 大小写敏感

  7. slice(fromIndex, toIndex) substr(fromIndex, toIndex) substring(fromIndex, toIndex) 截取字符串中的一部分，并作为返回值

    • 如果没有参数，默认为截取所用字符串内容

    • 如果只设定了fromIndex，toindex 没有设定，默认为截取到字符串结束位置

      不包含toindex指针对应的值

  8. split(separator,howmany) 分割字符串，返回数组

    • separator 表示用什么标记来分割，参数未设置，默认为不分割。

    • howmany 表示分割后的数组截获多少个值后，作为返回数组的值

  9. toLowerCase() toUpperCase() toLocalLowerCase() toLocalUpperCase() 没有参数，大小写切换

  10. valueOf() 获取原始字符串值，没有参数

ES6 - 字符串新特性

  1. includes() startsWith() endsWith()
  2. repeat()
  3. padStart() padEnd()

Array 数组

array - 数组创建

  1. new Array() 可以指定数组长度，或者列举出说有元素，作为参数传入
  2. let arr = [] 直接赋值

array - 数组操作

  1. concat() 连接数组，并返回结果数组。

  2. join() 将数组内元素以制定分隔符连接，返回字符串。

  3. push(value1, value2...) 在数组末尾添加元素，可多个。返回结果数组的长度值。

  4. pop() 在数组末尾删除一个元素，一次只能删除一个，没有参数。返回被删除元素值。

  5. reverse() 数组逆序操作，返回逆序数组。

  6. shift() 删除数组开头的一个元素，一次只能删除一个，没有参数。返回被删除元素值。

  7. unshift(value1, value2...) 在数组开头添加元素，可多个。返回结果数组的长度值。

  8. slice(fromIndex, toIndex) 截取数组中部分值。返回截取出元素组成的新数组。

    • 不包含toIndex

  9. sort() 对数组中元素进行排序。

    • sort() 排序后，原数组也被变更了。

    • 如果不传参，按照数组字符的编码顺序排序

    • 还可以传比较函数，比较函数返回一个数值与0进行比较，然后决定是否互换两个值的位置。

  10. splice(fromIndex, howmany, replacevalue) 增加或者删除数组元素，返回值是数组。

    • splice() 方法后，原数组也被改变了。

    • 如果是添加，返回值为空数组 [] 。

    • 如果是删除，返回值为被删去的元素组成的数组。

  11. valueOf() 没有参数，返回原始数组。

ES6 - 数组新特性

  1. ...arg 结构赋值
  2. find() findIndex()
  3. fill()
  4. entries() keys() values()
  5. includes()
